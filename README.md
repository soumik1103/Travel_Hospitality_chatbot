This project is an Intent-based AI Chatbot built using Natural Language Processing and a Deep Neural Network.

# Key Features:

Text preprocessing (tokenization, lemmatization)

Bag-of-Words vectorization

Intent classification

Neural Network with Dense layers

Dropout for regularization

SGD optimizer with momentum

Model saving and loading

Randomized responses from predicted intent

# Model Architecture

Input Layer → Bag-of-Words representation

Hidden Layers → Dense + Dropout

Output Layer → Softmax activation

Loss Function → Categorical Crossentropy

Optimizer → SGD

NLP preprocessing pipeline

Lemmatization using WordNet

Intent classification system
